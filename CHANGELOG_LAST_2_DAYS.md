# Changelog (последние 2 дня)

- Добавили обмен и слияние симбиозов: текстовый обмен, QR‑код, ручное поле, предупреждения о необратимости и подсказки про буфер обмена.
- Подправили логику роста: ограничения для отростков, приоритеты раннего роста тела, одно действие за цикл мутации и точнее выбор целей при росте.
- Улучшили визуал: фон сцены, подсветку, анимацию ветра для отростков, дыхание, внешние формы (например, морковь), контуры выбора.
- Настроили офлайн‑поведение: анабиоз после долгого отсутствия, лимиты и стоимость для мутаций, критические уведомления.
- Обновили интерфейс: подсказки и оверлеи, фиксы в легенде, вывод статистики (GPU/CPU/FPS), иконка проекта.
- Исправили ошибки и добавили запасные варианты кодирования данных для надёжного обмена.
